# Unreal Engine project — Age of the Apes

This folder will hold the UE5 project. Per [PRODUCTION_PLAN.md](../PRODUCTION_PLAN.md),
the gameplay is rebuilt here from the JS prototype (which serves as the design spec).

> **Why no `.uproject` yet?** The project must be generated by the Unreal Editor from the
> **Third Person** template — it can't be hand-authored. Follow Phase 0 below, point the
> editor at this `unreal/` folder, and the engine fills in `Config/`, `Content/`, `Source/`,
> and `AgeOfTheApes.uproject`. The repo is already set up to ignore the junk and LFS-track the assets.

---

## Phase 0 — one-time setup

### 1. Git LFS (do this BEFORE adding binary assets)
`git-lfs` is not installed on this machine yet. Install + enable:
```bash
brew install git-lfs        # macOS
git lfs install             # once per machine
```
The repo's [`.gitattributes`](../.gitattributes) already declares which file types go to LFS
(`.uasset`, `.umap`, `.glb`, textures, audio…). New matching files route to LFS automatically.

> The 4 ape GLBs already committed in `assets/models/` predate LFS. Optional migration:
> `git lfs migrate import --include="*.glb"` (rewrites history — coordinate if anyone else has cloned).

### 2. Install Unreal Engine
- Epic Games Launcher → install **UE 5.5 or newer** (5.5+ has native glTF/GLB import; 5.7 makes
  drag-drop trivial). On Apple Silicon, 5.5+ runs natively.

### 3. Create the project here
1. Editor → **New Project → Games → Third Person**.
2. Settings: **Blueprint** (C++ optional later), **Desktop**, Starter Content **off** (we bring our own look).
3. Name it `AgeOfTheApes`, location = this `unreal/` folder.
4. Enable plugins: **glTF Importer** / **Interchange** (Edit → Plugins), restart.

### 4. Verify import
- Drag `../assets/models/chimpanzee.glb` into the Content Browser. It should import as a Static Mesh.
  (Rigged/animated GLBs come later in Phase 3 via Mixamo.)

### 5. First commit
```bash
git add unreal/AgeOfTheApes.uproject unreal/Config unreal/Content unreal/Source 2>/dev/null
git commit -m "Add UE5 Third Person project scaffold"
```

When Phase 0 passes, move to **Phase 1** → [docs/toon-shader-setup.md](docs/toon-shader-setup.md).

## Folder convention (inside `Content/`)
```
Content/
  AOA/                # all our stuff under one root to keep it separate from template
    Characters/       # apes (hero + NPC species)
    Environments/     # zones: HumanRuins / TradePost / ApeForest
    Materials/        # M_Toon master + instances, emission, fog
    Props/            # generated props, foliage, bridges, fences
    Blueprints/       # gameplay: character, camera, combat, dialog, quests
    UI/               # HUD, inventory, dialog widgets
```
